
# 🎓 עוזר הלימודים החכם | AI Learning Center

[![Vercel Deployment](https://img.shields.io/badge/Vercel-Deployed-success?style=for-the-badge&logo=vercel)](https://learning-center-by-gemini.vercel.app/)
[![Powered by Gemini](https://img.shields.io/badge/Powered%20By-Google%20Gemini-blue?style=for-the-badge&logo=google-gemini)](https://ai.google.dev/)

**העוזר החכם שלכם למבחנים** – פלטפורמה אינטראקטיבית ללמידה מבוססת תוכן אישי וקהילתי. המערכת מאפשרת להעלות קבצי לימוד, תרגילים וסיכומים, ומספקת מענה מדויק על סמך התוכן שהועלה בלבד, תוך דגש על איכות שפה עברית גבוהה ונוסחאות מתמטיות מרהיבות.

---

## ✨ תכונות עיקריות (Core Features)

### 📚 מאגר ידע גלובלי (Shared Knowledge)
כל המשתמשים במערכת חולקים את אותן "תיקיות ידע". העליתם סיכום מצוין ב"מבנים אלגבריים 2"? עכשיו כולם יכולים להיעזר בו.
* **Algebraic Structures 2**: תיקיית ברירת מחדל מוכנה הכוללת סיכומים ותרגילים.

### 🤖 צ'אט מבוסס הקשר (Context-Aware Chat)
ה-AI לא "מנחש". הוא קורא את הקבצים שהעליתם ומצטט מתוכם.
* **הפניות למקורות**: כל תשובה כוללת ציטוט מהקובץ הרלוונטי.
* **אימות עברית**: כל מענה עובר עיבוד נוסף כדי להבטיח עברית טבעית ולא "רובוטית".

### 📐 תמיכה מלאה במתמטיקה
תצוגת נוסחאות LaTeX מורכבות בצורה ברורה וחדה (בצבע לבן על רקע כהה לנוחות הקריאה).
* פונקציות, קבוצות, חבורות וסמלים מורכבים.

### 🔍 חיפוש מתקדם
חיפוש מהיר בתוך שמות התיקיות, שמות הקבצים ואפילו בתוך התוכן הטקסטואלי של קבצי ה-MD וה-TXT שהועלו.

### 👤 היסטוריה אישית ומצב אורח
* **מצב אורח**: למידה מיידית ללא רישום.
* **חשבון אישי**: שמירת היסטוריית הצ'אטים וסנכרון בין מכשירים.

## 🛠 טכנולוגיות (Tech Stack)

* **Frontend**: React 19 + Tailwind CSS.
* **AI Engine**: Google Gemini 3 (flash-preview).
* **Rendering**: KaTeX (Math), Lucide (Icons).
* **Language Logic**: Custom Hebrew Validation Service.
